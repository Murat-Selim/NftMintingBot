# NftMintingBot

# Getting Started

To run this bot, mint Nfts

## Prerequisites

- Python 3.6+
- pip install
  
```
pip install web3
```
- `os` library
- `load_dotenv` library

## Built With

[Web3.py ](#https://web3py.readthedocs.io/en/stable/)- Ethereum API for Python
